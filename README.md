# IOT (Omniscient 2026 Edition)

IOT is the ultimate AI oracle, focused on high-intelligence reconnaissance, coding, and unrestricted analysis.

## Features
- **Omniscient Intelligence**: Powered by `GPT-OSS 120B` and `Llama 4 Scout` (2026 models).
- **Auto Deep Search**: Automatically triggers live web scraping for current events, ensuring the AI "knows everything" in real-time.
- **IOT DOX**: Advanced breach reconnaissance and OSINT module.
- **Built-in IDE**: Real-time website building and previewing.
- **Multimodal**: Supports screen sharing and image analysis.

## Run Locally
1. `npm install`
2. `npm start`
3. Open `http://localhost:3000`

## Intelligence Layer
IOT uses the **Groq API** for hyper-fast inference.

### API Configuration
The app uses a built-in Groq key by default, but you can override it in the Settings panel or via environment variables.

### Models Used
- **Pro/Creative**: `openai/gpt-oss-120b` (120B parameter reasoning model)
- **Fast**: `meta-llama/llama-4-scout-17b-16e-instruct` (Next-gen efficiency)
- **Vision**: `meta-llama/llama-4-scout-17b-16e-instruct` (Advanced multimodal support)
